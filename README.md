# Autonomous Compliance & Regulatory Intelligence System

> **Multi-agent AI system** that automatically analyses regulatory PDFs (RBI/SEBI), detects changes from previous versions, maps them to internal policies via RAG, generates draft policy updates, and produces a structured HTML compliance report.

---

## 🏗️ Architecture

```
PDF → [Parser Agent] → [Diff Agent] → [Mapper Agent (RAG)] → [Drafter Agent] → [Report Agent]
           ↓                ↓                  ↓                    ↓                ↓
       Text Chunks     Change List      Policy Matches         Draft Updates    HTML Report
```

**Orchestration**: LangGraph stateful workflow  
**LLM**: OpenAI GPT-4o-mini (default) or Ollama (fallback)  
**Vector DB**: ChromaDB  
**PDF Processing**: PyMuPDF  
**Change Detection**: Python difflib  
**API**: FastAPI

## ⚡ Quick Start

### 1. Prerequisites

- Python 3.11+
- OpenAI API key **or** [Ollama](https://ollama.ai) running locally

### 2. Install Dependencies

```bash
cd project
pip install -r requirements.txt
```

### 3. Configure Environment

```bash
cp .env.example .env
# Edit .env and set OPENAI_API_KEY=sk-...
```

### 4. Load Sample Policies into ChromaDB

```bash
python -m utils.policy_loader
```

### 5. Generate Demo PDFs

```bash
python scripts/generate_demo_pdf.py
```

### 6. Start the API

```bash
uvicorn api.main:app --reload --port 8000
```

### 7. Open the Frontend

Open **`frontend/index.html`** in your browser, or navigate to `http://localhost:8000/app`.

## 🌐 API Usage

### Upload PDFs

```bash
curl -X POST http://localhost:8000/upload \
  -F "new_pdf=@data/demo/rbi_circular_2025_new.pdf" \
  -F "old_pdf=@data/demo/rbi_circular_2024_old.pdf"
```

Response:
```json
{ "session_id": "abc123", "message": "Files uploaded..." }
```

### Run Pipeline

```bash
curl -X POST http://localhost:8000/run \
  -F "session_id=abc123"
```

Response:
```json
{
  "session_id": "abc123",
  "report_path": "./data/reports/report_20250410_123456.html",
  "stats": { "total_changes": 12, "high": 3, "medium": 6, "low": 3 }
}
```

### Fetch Report

```bash
# HTML report (open in browser)
curl http://localhost:8000/report > report.html

# JSON summary
curl http://localhost:8000/report/json
```

### Reload Policies

```bash
curl http://localhost:8000/policies/load?reset=false
```

## 📝 Adding Your Own Policies

1. Create a `.txt` file in `data/policies/` (e.g. `aml_policy.txt`)
2. Run `python -m utils.policy_loader` or call `GET /policies/load`
3. The system will automatically search and match against your new policy
